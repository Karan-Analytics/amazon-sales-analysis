# Amazon Sales Analysis Dashboard

## Overview

This project analyzes Amazon sales data using Microsoft Excel, MySQL, and Power BI. The workflow includes data cleaning, SQL analysis, and interactive dashboard creation.

## Tools Used

* Microsoft Excel
* MySQL
* Power BI

## Project Workflow

* Data Cleaning in Excel
* SQL Business Queries
* Power BI Dashboard Development
* Business Insights & Visualization

## Key Insights

* Total Revenue
* Total Orders
* Revenue by Category
* Revenue by City
* Order Status Analysis
* Fulfilment Analysis
* Courier Status Analysis

## Files Included

* Amazon E-commerce Sales Dataset
* SQL Queries
* Power BI Dashboard (.pbix)
* Dashboard Screenshots
